# FoodOrdringApp

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 7.0.2.

## Development server

```
$ git clone https://github.com/AbderrahimSoubaiElidrissi/food-angular-app
$ cd food-angular-app 
$ npm install
$ ng serve
```
